<h1 align="center">xhs-comment-collector</h1>

This app allows you to collect Xiaohongshu comments, supporting categorization, search, export, pinning, batch management, tag and note management, image and text parsing, and data statistics.

此应用可收集小红书评论，支持分类、搜索、导出、置顶、批量管理、标签和笔记管理、图文解析以及数据统计。
